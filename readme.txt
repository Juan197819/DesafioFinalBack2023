Todas las routers de la carpeta "routes" se pueden probar desde POSTMAN o desde la URL desde el navegador. Para probar el "routerSessions" se puede acceder desde localhost:8080 y registrandose
(hacerlo con mail real para probar la funcion de envio de mails para la compra del carrito). Si el usuario es "admin" las vistas creadas mostraran solo los botones para administradores 
y si es cualquier otro sera"user" y tendra botones para seleccionar productos y comprar carritos (botones que el usuario "admin" no tiene) pero no los botones para crear o modificar productos 
que si tiene el "admin" (por defecto se dejo "adminCoder@coder.com" como unico "admin" pero este mail puede setearse a gusto ya que se saco del archivo .env como variable de entorno).
